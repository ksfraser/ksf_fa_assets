## 1. UAT Objectives

Verify that:
1. Assets display correctly in all views
2. Serial number lookup finds correct assets
3. Maintenance history shows complete records
4. Upcoming maintenance list is accurate

| UAT-003: Lookup by serial - not found | Error message shown | Asset Manager |
| UAT-004: View maintenance history | History table displayed | Asset Manager |
| UAT-005: View upcoming maintenance | Items due within 30 days | Maintenance Staff |
